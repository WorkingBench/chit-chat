## General info
This is a personal project built with ExpressJS, SocketIO & ReactJs.
	
## Setup
To run this project, install it locally using npm and yarn:

* For client (React App)
```
cd client
yarn start
```
* For server (NodeJS App)
```
cd server
npm run dev
```
